Introducción

Linux Mint es un sistema operativo pensado para trabajar en la mayoría de sistemas modernos, es una distribucion de Linux basada en Debian o en Ubuntu(dependiendo de la versión), que busca ser moderna, sencilla, rápida y potente. Es un sistema operativo que se comenzó a desarrollar en 2006, en una versión beta llamada Ada, su principal objetivo era ser de fácil uso, gracias a esto, ha conseguido colocarse en los puestos más altos de popularidad de distribuciones Linux. Mint tiene como objetivo proporcionar a usuarios, empresas, compañías un software que puedan utilizar sin coste alguno.

Información general

El equipo de desarrollo de Linux Mint está formado por un equipo de 20 desarroladores liderados por Clement Lefevre, los cuales estan financiados por la comunidad a través de donaciones, patrocinios, colaboraciones y publicidad. Es un software libre y de código abierto, concretamente bajo licencia GNU GPL(General Public License), quedando disponible para el usuario el uso y modificación del software. Incluye el mismo kernel de Linux (Monolítico), las herramientas GNU y diferentes entornos de escritorio que veremos más adelante. Utiliza dpkg como sistema de gestión de paquetes y permite el soporte para las plataformas x86 (32 bits), x86_64 (64 bits) e Itanium IA-32.

Ediciones

Según el entorno de escritorio que se utilice, a día de hoy Linux Mint está disponible en 3 ediciones distintas:

  • Cinnamon: Es la edición principal de Linux Mint, con un escritorio más moderno, innovador y completo. Cinnamon es un entorno de escritorio desarrollado por el equipo de Linux Mint basado en el shell de GNOME.
  • MATE: Aunque carece de algunas caracteísticas como las de Cinnamon, es un entorno más veloz y estable. Fue desarrollado por la comunidad y con el apoyo del equipo de Linux Mint despues de que el cambio de GNOME 2 a GNOME 3 produjera problemas en el rendimiento.
  • XFCE: XFCE es la versión más ligera y estable de Linux Mint con un entorno más minimalista, no soporta tantas características como MATE o Cinnamon y está pensado para ser instalado en los equipos menos potentes.

A mayores, la primera versión de Linux Mint utilizó KDE como entorno de escritorio y lo siguió utilizando hasta ser sustituido por GNOME entre 2008 y 2011.

Repositorios

Linux Mint utiliza diferentes repositorios propuestos por los desarrolladores en su página de GitHub , cada uno de estos repositorios contiene paquetes que pueden ser instalados fácilmete en el sistema operativo. Algunos de lo reposistorios más populares son:

  1. Cinnamon-spices-applets: Este repositorio contiene todos los subprogramas disponibles para la edición Cinnamon.
  2. Warpinator: Utilizado para compartir archivos a través de la red LAN.
  3. Nemo: Gestor de archivos para Cinnamon.
  4. Nemo-extensions: Contiene todas las extensiones disponibles para nemo
  5. Hypnotix: Es una aplicación que soporta transmisión de TV en directo, series y películas.
  6. Sticky: Contiene un programa para tomar notas en el escritorio.
  7. Muffin: Gestiona las ventanas del escritorio Cinnamon.
  8. Xreader: Es un visor de documentos capaz de abrir archivos de múltiples formatos.
  9. Cjs: Interprete de javascript de Cinammon.

Herramientas preinstaladas

Linux Mint nos proporciona una serie de herramientas básicas que vienen preinstaladas para que el usuario pueda utilizar el sistema nada más concluir la instalación, algunos de estos programas son:

  • GIMP: Como editor gráfico.
  • LibreOffice: Como herramienta de ofimática.
  • Mozilla Firefox: Como navegador web.
  • Mozilla Thunderbird: Como cliente de correo electrónico.
  • Transmission: Como cliente de BitTorrent.
  • Rhythmbox: Reproductor y organizador de música.
  • Simple-scan: Herramienta de escaneo.
  • Timeshift: Herramientapara realizar backups del sistema.
  • Xef: Como editor de texto predeterminado.
  • Gdb: Debugger de GNU
  • Pix: Visor y buscador de imágenes.

Además, tambien nos implementa una serie de aplicaciones para hacer más sencilla la experiencia del usuario.

  • MintUpdate: Es el encargado de buscar, bajar e instalar las actualizaciones Asigna a cada actualización un nivel de seguridad puntuado de 1 a 5 siguiendo criterios de estabilidad y necesidad de actualización.
  • MintInstall: Sirve para instalar programas desde el catálogo que se encuentra en el Portal de Software de Linux Mint.
  • MintUpload: Facilita la compartición de archivos.
  • MintConfig: Es un centro enfocado a la configuración del sistema.
  • MintBackup: Facilita el respaldo y restaurción de archivos de usuario y software del sistema.
  • MintDesktop: Utilizado para la confiuguración del escritorio.
  • MintAssistant: Un asistente que aparece durante el primer acceso del usuario, guiándole para personalizar la base de Mint.
  • MintMenu: Es un menú que permite la personalización de textos, iconos y colores.
  • MintNanny: Es un programa que permite restringir el acceso a ciertas páginas de internet definidas por el usuario.

Conclusiones

Linux Mint tiene el objetivo de proveer de un sistema operativo de escritorio sencillo, eficiente y gratis a usuarios de hogar y compañías que lo necesiten para realizar sus tareas habituales, en otras palabras, está enfocado a la productividad. Para ello está dotado de una gran cantidad de herramientas y es compatible con muchos programas. Es un sistema operativo muy rápido, estable y con buena seguridad protegiendo la privacidad frente a ataques maliciosos. Además de todo esto, Linux Mint es muy personalizable.
